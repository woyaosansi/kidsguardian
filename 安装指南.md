# 少儿卫士 App 安装指南（详细版）

> 检测到您的系统已安装 Java JDK 14.0.1，但仍需要安装 Android Studio 来获取 Android SDK 和 Gradle 构建工具。本指南将一步步带您完成安装。

---

## 总览：安装流程

```
第1步：安装 Android Studio（含 Android SDK）
   ↓
第2步：用 Android Studio 打开项目
   ↓
第3步：等待 Gradle 自动同步和下载依赖
   ↓
第4步：手机开启 USB 调试模式
   ↓
第5步：连接手机到电脑
   ↓
第6步：点击运行按钮，自动编译并安装到手机
```

## 第1步：安装 Android Studio

### 1.1 下载 Android Studio

1. 打开浏览器，访问官方下载页面：
   ```
   https://developer.android.com/studio
   ```
2. 找到 **"Android Studio Giraffe | 2022.3.1"** 或更新版本
3. 点击 **"Download Android Studio"** 按钮
4. 勾选 **"I have read and agree to the above terms and conditions"**
5. 点击 **"Download Android Studio"** 开始下载

> 文件大小约 1GB，请耐心等待下载完成。

### 1.2 安装 Android Studio

1. 找到下载的安装文件（如 `android-studio-2022.3.1.x-windows.exe`）
2. **双击运行**安装程序
3. 安装向导启动后，按以下步骤操作：

   | 步骤 | 操作 |
   |------|------|
   | Welcome | 点击 **Next** |
   | Choose Components | 保持默认，点击 **Next** |
   | Configuration Settings | 保持默认路径，点击 **Next** |
   | Choose Start Menu Folder | 点击 **Install** |
   | Installing | 等待安装完成 |
   | Completing | 勾选 "Start Android Studio"，点击 **Finish** |

4. 首次启动 Android Studio 时：
   - 选择 **"Do not import settings"**（不导入设置）
   - 点击 **OK**
   - 选择 **"Standard"** 标准安装
   - 点击 **Next**，然后点击 **Finish**
   - 等待组件下载完成（这一步会自动下载 Android SDK）

> ⏱️ 此步骤可能需要 10-30 分钟，取决于网络速度

---

## 第2步：用 Android Studio 打开项目

### 2.1 打开项目

1. 启动 Android Studio
2. 在欢迎界面点击 **"Open"**（或菜单栏 File → Open）
3. 在文件选择器中导航到：
   ```
   d:\AliWangWang\少儿卫士
   ```
4. 选中 **"少儿卫士"** 文件夹
5. 点击 **OK**

### 2.2 等待 Gradle 同步

打开项目后，Android Studio 底部会显示 **"Gradle sync in progress..."**：

1. 此时 Android Studio 正在自动下载：
   - Gradle 构建工具
   - Kotlin 插件
   - AndroidX 库
   - 其他依赖项
2. 等待底部状态栏显示 **"Gradle sync finished"**
3. 如果弹出提示选择 SDK，请点击 **"Yes"** 安装

> ⏱️ 首次同步可能需要 5-15 分钟

### 2.3 如果 Gradle 同步失败

如果底部出现红色错误提示，请尝试以下方法：

**方法1：配置国内镜像源（推荐）**

打开 `build.gradle` 文件，将 repositories 部分修改为：

```gradle
allprojects {
    repositories {
        maven { url 'https://maven.aliyun.com/repository/google' }
        maven { url 'https://maven.aliyun.com/repository/public' }
        google()
        mavenCentral()
    }
}
```

然后点击 **"File → Sync Project with Gradle Files"** 重新同步。

**方法2：检查代理设置**

如果网络需要代理，请配置：
1. File → Settings → Appearance & Behavior → System Settings → HTTP Proxy
2. 设置代理服务器
3. 重新同步项目

## 第3步：手机设置

### 3.1 开启开发者选项

在手机上操作：

1. 打开 **设置**
2. 找到 **关于手机**（路径因手机品牌而异）：
   - 小米/红米：设置 → 我的设备 → 全部参数
   - 华为/荣耀：设置 → 关于手机
   - OPPO/vivo：设置 → 关于手机
   - 三星：设置 → 关于手机 → 软件信息
3. 找到 **版本号**（或 "MIUI版本"、"内部版本号"）
4. **连续快速点击 7 次**
5. 看到 **"您已处于开发者模式"** 的提示

### 3.2 开启 USB 调试

1. 返回 **设置** 主页面
2. 找到 **开发者选项**：
   - 小米：设置 → 更多设置 → 开发者选项
   - 华为：设置 → 系统和更新 → 开发人员选项
   - OPPO：设置 → 其他设置 → 开发者选项
3. **打开开发者选项开关**
4. 找到 **"USB调试"** 并开启
5. 如果有以下选项也建议开启：
   - **"USB安装"**（允许通过USB安装应用）
   - **"仅充电模式下允许USB调试"**
   - **"禁止权限监控"**

### 3.3 检查 USB 连接模式

1. 用 **原装数据线** 连接手机和电脑
2. 手机上弹出 USB 连接方式选择，选择 **"传输文件 (MTP)"**
3. 如果没有弹出，从顶部下拉通知栏，点击 USB 通知

---

## 第4步：连接并验证手机

### 4.1 在电脑上验证

1. 打开 Android Studio
2. 查看顶部工具栏的设备下拉列表：
   ```
   [📱 手机型号] ▼   [app ▼]   ▶ 运行
   ```
3. 如果看到您的手机型号，说明连接成功

### 4.2 首次连接的授权

1. 连接后手机上会弹出：
   ```
   允许 USB 调试吗？
   该计算机的 RSA 密钥指纹如下：...
   ☑ 一律允许使用这台计算机进行 USB 调试
   ```
2. **勾选 "一律允许"**
3. 点击 **"确定"** 或 **"允许"**

### 4.3 如果没有看到设备

请检查以下问题：

| 问题 | 解决方案 |
|------|----------|
| 数据线问题 | 使用原装线，有些线只能充电不能传数据 |
| 驱动问题 | Windows → 设备管理器 → 查看是否有黄色感叹号 |
| USB模式问题 | 手机选择 "传输文件" 而非 "仅充电" |
| 调试未开启 | 重新检查开发者选项中的 USB 调试 |

## 第5步：编译并安装到手机

### 5.1 一键安装

1. 在 Android Studio 顶部工具栏：
   - 确认左侧下拉框选择了 **您的手机设备**
   - 确认中间下拉框选择了 **app**
   - 确认右侧模块选择 **app**

2. 点击绿色的 **▶ 运行按钮**（或按 **Shift + F10**）

3. Android Studio 开始：
   ```
   Executing tasks: [:app:assembleDebug, :app:installDebug]...
   ```

4. 等待编译完成（首次约需 2-5 分钟）

5. 编译成功后，手机上会自动安装并打开应用

### 5.2 安装后授权

应用首次启动时，需要授权以下权限：

**① 悬浮窗权限**（最重要）
```
手机弹出提示 → 点击"设置" → 开启"允许显示在其他应用上层" → 返回
```

**② 通知权限**
```
设置 → 应用 → 少儿卫士 → 通知 → 允许
```

**③ 后台运行权限**（部分手机需要）
```
设置 → 应用 → 少儿卫士 → 电池/省电 → 允许后台运行
```

## 备用方法：生成 APK 手动安装

如果直接运行安装失败，可以先生成 APK 文件再手动安装。

### 方法A：通过 Android Studio 生成 APK

1. 菜单栏：**Build → Build Bundle(s) / APK(s) → Build APK(s)**
2. 等待编译完成
3. 编译成功后，点击通知 **"locate"** 找到 APK 文件
4. APK 文件路径：
   ```
   d:\AliWangWang\少儿卫士\app\build\outputs\apk\debug\app-debug.apk
   ```

### 方法B：将 APK 传到手机安装

1. 用数据线、微信、QQ、邮箱等方式将 APK 文件传到手机
2. 在手机文件管理器中找到 APK 文件
3. 点击安装
4. 如果提示 "未知来源"，按提示允许安装

## 第6步：使用应用

### 6.1 启动保护

1. 打开 "少儿卫士" 应用
2. 阅读 UI 上的说明文字
3. 点击 **"开始保护"** 按钮
4. 按钮文字变为 **"停止保护"**，状态显示绿色 "保护已开启"
5. 应用开始后台运行，通知栏会显示运行状态

### 6.2 测试功能

1. **正常使用**：坐直玩手机 → 不会有提醒
2. **测试躺着**：躺下，将手机举到面部上方（屏幕朝下看） → 触发提醒
3. 触发提醒时：
   - 发出 "嘀嘀嘀" 声音
   - 手机震动
   - 弹出提醒界面
4. 点击 **"我知道了"** 关闭提醒

### 6.3 停止保护

1. 打开应用
2. 点击 **"停止保护"** 按钮
3. 按钮文字变为 **"开始保护"**，状态显示红色 "保护已关闭"

## 常见问题解答

### Q: Gradle 同步很慢或失败怎么办？

**A:** 配置阿里云镜像源加速下载：
1. 打开项目根目录的 `build.gradle`
2. 在 `allprojects` → `repositories` 中添加：
   ```gradle
   maven { url 'https://maven.aliyun.com/repository/google' }
   maven { url 'https://maven.aliyun.com/repository/public' }
   ```
3. 放在 `google()` 和 `mavenCentral()` 之前
4. 重新同步

### Q: 手机上不显示在设备列表中怎么办？

**A:** 依次检查：
1. 确认使用了 **数据线**（不是纯充电线）
2. 手机上选择了 **"传输文件"** 模式
3. 已开启 **USB 调试**
4. 安装了手机的 **USB 驱动程序**
5. 重新插拔 USB 线
6. 尝试其他 USB 端口

### Q: 安装时提示 " INSTALL_FAILED_VERIFICATION_FAILURE " 怎么办？

**A:** 这是手机安全限制：
1. 设置 → 安全 → 允许安装未知来源应用
2. 或在开发者选项中开启 **"USB 安装"**

### Q: 应用闪退或不工作怎么办？

**A:** 
1. 确认已授予 **悬浮窗权限**
2. 确认已开启 **通知权限**
3. 在 Android Studio 中查看 **Logcat** 日志排查问题
4. 确认手机有加速度传感器（一般都有）

### Q: 检测不准确怎么办？

**A:** 可以修改 `PostureDetectionService.kt` 中的参数：
- `LYING_THRESHOLD`：调小更敏感（如 0.5），调大不敏感（如 0.85）
- `ALERT_THRESHOLD`：调小更快触发（如 2），调大更慢触发（如 5）
