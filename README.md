# 少儿卫士 - 少儿视力保护应用

## 应用介绍
少儿卫士是一款专为保护少年儿童视力健康而设计的安卓应用。当检测到用户躺着玩手机时，应用会发出声音提醒，并可以选择自动锁屏来保护用户的视力。

## 主要功能

### 1. 姿态检测
- 利用手机加速度传感器检测用户使用手机的姿势
- 智能识别躺着玩手机的场景
- 可自定义检测灵敏度

### 2. 声音提醒
- 当检测到不良姿势时发出滴滴声
- 连续三次提醒以引起注意
- 声音清晰且温和

### 3. 自动锁屏
- 可设置是否启用自动锁屏功能
- 检测到不良姿势后自动暂时锁屏
- 支持自定义锁屏时长

### 4. 状态监控
- 实时显示保护服务状态
- 后台持续监控
- 低功耗运行

## 技术架构

### 核心组件
- **MainActivity**: 主界面，提供开始/停止保护功能
- **PostureDetectionService**: 后台服务，负责姿态检测
- **AlertActivity**: 提醒界面，显示警告信息
- **ScreenReceiver**: 屏幕状态监听

### 使用技术
- Android Sensor API (加速度传感器)
- Service 和 Foreground Service
- WakeLock 电源管理
- ToneGenerator 声音生成
- Vibrator 震动提醒

## 使用说明

1. 打开应用
2. 点击"开始保护"按钮
3. 应用自动开始检测您的使用姿势
4. 当检测到躺着玩手机时，会发出声音提醒
5. 如需停止保护，点击"停止保护"按钮

## 权限说明

应用需要以下权限：
- **WAKE_LOCK**: 保持设备唤醒状态
- **DISABLE_KEYGUARD**: 管理锁屏
- **SYSTEM_ALERT_WINDOW**: 显示提醒界面
- **FOREGROUND_SERVICE**: 后台服务运行
- **USE_FULL_SCREEN_INTENT**: 全屏提醒

## 安装和运行

1. 使用Android Studio打开项目
2. 配置Gradle和依赖
3. 连接安卓设备或模拟器
4. 点击运行按钮

## 自定义设置

您可以通过修改以下参数来自定义应用行为：
- `LYING_THRESHOLD`: 躺着检测阈值 (0.0-1.0)
- `DETECTION_INTERVAL`: 检测间隔 (毫秒)
- `ALERT_THRESHOLD`: 提醒阈值次数

## 注意事项

1. 建议在充电时使用，以免影响正常使用
2. 应用需要保持在前台或后台运行
3. 不同手机的传感器可能略有差异
