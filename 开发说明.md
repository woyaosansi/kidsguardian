# 少儿卫士应用开发说明

## 项目概述
这是一个完整的安卓应用开发项目，旨在保护少年儿童的视力健康。应用通过检测手机姿态来判断用户是否躺着玩手机，并在检测到不良姿势时进行提醒。

## 核心功能实现

### 1. 姿态检测算法
应用使用手机的加速度传感器来检测用户的使用姿势：

```kotlin
private fun detectPosture(accelValues: FloatArray) {
    val x = accelValues[0]
    val y = accelValues[1]
    val z = accelValues[2]
    
    val magnitude = sqrt(x * x + y * y + z * z)
    val normalizedZ = abs(z) / magnitude
    
    if (normalizedZ > LYING_THRESHOLD) {
        // 检测到躺着使用手机
        triggerAlert()
    }
}
```

### 2. 声音提醒系统
使用 ToneGenerator 产生连续的三段提醒声音：

```kotlin
private fun playAlertSound() {
    toneGenerator.startTone(ToneGenerator.TONE_CDMA_ALERT_CALL_GUARD, 500)
    Thread.sleep(600)
    toneGenerator.startTone(ToneGenerator.TONE_CDMA_ALERT_CALL_GUARD, 500)
    Thread.sleep(600)
    toneGenerator.startTone(ToneGenerator.TONE_CDMA_ALERT_CALL_GUARD, 500)
}
```

### 3. 智能锁屏机制
当检测到不良姿势时，暂时禁用屏幕交互：

```kotlin
private fun lockScreen() {
    val params = WindowManager.LayoutParams().apply {
        type = WindowManager.LayoutParams.TYPE_SYSTEM_ERROR
        flags = WindowManager.LayoutParams.FLAG_NOT_FOCUSABLE or
                WindowManager.LayoutParams.FLAG_NOT_TOUCHABLE
    }

}
```

## 关键配置参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `LYING_THRESHOLD` | 0.7 | 躺着检测阈值 |
| `DETECTION_INTERVAL` | 2000ms | 检测间隔 |
| `ALERT_THRESHOLD` | 3 | 提醒阈值次数 |

## 运行要求

### 最低要求
- Android API Level 24 (Android 7.0)
- 目标SDK版本 34
- Kotlin 1.8.22
- Android Studio Giraffe 或更高版本

### 必要权限
应用需要以下权限才能正常运行：
- `android.permission.WAKE_LOCK` - 保持设备唤醒
- `android.permission.DISABLE_KEYGUARD` - 管理锁屏
- `android.permission.SYSTEM_ALERT_WINDOW` - 显示提醒覆盖层
- `android.permission.FOREGROUND_SERVICE` - 前台服务
- `android.permission.USE_FULL_SCREEN_INTENT` - 全屏意图

## 调试技巧

### 常见问题和解决方案

1. **传感器不工作**
   - 确保设备支持加速度传感器
   - 检查权限是否已授予
   - 验证传感器注册是否成功

2. **前台服务被杀死**
   - 确保正确设置了前台服务通知
   - 检查电池优化设置
   - 验证 WakeLock 是否正确获取

3. **锁屏功能无效**
   - 确认 SYSTEM_ALERT_WINDOW 权限
   - 检查 Android 版本兼容性
   - 验证 WindowManager 使用方式
